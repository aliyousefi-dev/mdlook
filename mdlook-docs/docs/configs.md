# Configs

when you create a workstation. you can config that.

it exist on this path
`config.json` on workstation.

## Default Config

```json
{
	"docname": "MDLook",
	"disableThemes": true,
	"disablePrintOptions": true,
	"disableMarkdownOptions": true,
	"disableToC": false,
	"GitUrl": "https://github.com/user/repo"
}
```
