# Code Highlight

mdlook provides syntax highlighting for a broad selection of programming languages using the Shiki highlighter.

Below is a table of supported language grammars, including their unique IDs and common aliases.

For the most up-to-date list and further details, visit the [Shiki Supported Languages](https://shiki.style/languages) page.
